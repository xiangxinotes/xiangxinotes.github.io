作为一个爱做计划、脑海里有很多想法，但很少落地的人，我有时候晚上睡不着都在想：那些写在待办清单上的事，到底什么时候才能完成呀？没错，我还有点强迫症，一想到那么多事没干，就特别烦躁。

当然，除了烦躁，我还总批评自己，觉得自己什么都干不了，永远没出息。毕竟，光有想法有什么用呢？谁没有想法？真正改变世界、创造生活的，永远是付诸行动的人。

可惜，批评根本没用。骂得太狠，我反而会破罐子破摔，干脆躺平一阵子再说。

为了逼自己行动，我把 Bilibili 账号取名叫[**TakeAction_行动**](https://space.bilibili.com/329955183){:target="_blank"}，还跟自己约定：“等你真正做到持续行动了，再改名字。”我想，只要名字一天不改，我每次看到都会觉得羞耻，就会被迫动起来。无论如何，先行动就好。

结果，这个账号名一直用到现在，哈哈。

不过最近我有了新的预感：**2026 年，很可能就是我摆脱这个账号名的一年！**

原因很简单：

我最近碰巧看了 Ali Abdaal 的视频 —— [有目标没行动？“梦想家”们都在犯的3种错误](https://www.bilibili.com/video/BV1sfcEzUE4q/){:target="_blank"}。

看完之后，我结合视频内容和自己的经历，做了一套**梦想家行动指南**。

我把自己的时间表、计划清单重新梳理，专门留出固定时间段写博客，结果**只用 2 天，就写完了拖延 2.5 个月的文章**。

那种真正行动起来、把待办事项一条条划掉的释然，真的像重获新生。

## 你是梦想家吗？

Ali Abdaal 在视频里，把人分成四类：
- **流浪者（Drifters）**，随波逐流，没志向，没行动
- **梦想家（Dreamers）**，有志向，没行动
- **仓鼠人（Hamsters）**，没志向，有行动
- **大师（Masters）**，有志向，有行动

所以，梦想家 = **有目标、有想法、有理想，但一直原地踏步的人**。

梦想家们好像总爱在脑子里想一想，就满足了，有点像电影《白日梦想家》的主角。

我自己并不能从 “只做计划不行动” 里获得真正的满足。

做计划会带来一定的满足感，但如果我一直不行动，那么做计划的时候有多开心，没行动的时候看到这个计划就有多恼怒。

看到这里，你可以问问自己：“你是梦想家么？” “你是怎样的梦想家？”

## 梦想家行动指南

这份行动指南，是我看完 Ali Abdaal 的视频后，结合自己的理解和想法整理总结出来的。

我用它梳理了自己的四大目标，我发现自己分析之后变得非常有行动力。

因为我终于分清：哪些我是**真的想要**，哪些只是**人云亦云**。

同时，我也大胆划掉了很多不适合自己的事项，整个人轻松了一大截。

因为我感觉很好，所以把它分享给你，希望在我走向“大师”的路上也能看到你的身影。

### 第一步：找出你的时间表和计划清单

翻出你的最新时间表，和你做过的所有计划清单。

找不到全部的计划清单也没事，先把能找到的一两个拿在手里。

### 第二步：事项划分 —— 合并同类项

把所有事项，按主题归纳成几个大类。

比如：学习、健身、自媒体、技能、旅行、人际关系等。

### 第三步：针对每一个大类，一步步深度分析

拿一张空白 A4 纸，或在电脑上新建文档。

对**每一个大类**，按下面的流程完整写一遍。

#### ① 写下过去的目标

```
过去我以为想要【这里填入你的大类名称】
```

#### ② 写下你自以为的理由

问自己：

**“我为什么觉得自己想要【这里填入你的大类名称】？”**

把能想到的理由全部写下来，分成两类：
- 逻辑理由 1～N（理性上的理由）
- 情绪理由 1～N（内心真实的感受）

#### ③ 确定真心：你是真的想要吗？

再问自己：

**“我现在，真的还想要【这里填入你的大类名称】吗？”**

- 如果**只有逻辑理由，完全没有情绪理由**
  - → 你其实**根本不想要**，只是觉得 “应该要”。
  - → 建议你**把这个大类从计划里划掉**。
- 如果你舍不得删，但又想不出情绪理由
  - → 可以和 AI 聊聊，让它帮你挖一挖内心真实动机。
  - → 如果还是找不到，**哪怕暂时划掉**，也会让你轻松很多。
- 如果你确定：**我是真的想要**
  - → 继续往下走。

#### ④ 写下所有真实障碍

把**所有阻碍你行动的原因**全部写出来。

可以是现实问题，也可以是情绪、恐惧、完美主义。

重点：**一定要写真实障碍，不要骗自己。**

比如：
- 假障碍：没有好看的背景，没法做室内自媒体
- 真障碍：我害怕背景不够完美，不敢开始

#### ⑤ 分析障碍：哪些在你掌控中？

对每一个障碍问自己：

**这件事，在我的掌控范围内吗？**

只有三种答案：
- 不在我的掌控中
- 部分在我的掌控中
- 完全在我的掌控中

- 如果**完全不在掌控**
  - → 是现实硬限制，且无法改变
  - → 果断把这个大类**划掉**。
- 如果**部分在掌控**
  - → 放下不能控制的部分，只做你能控制的事。
- 如果**在掌控**
  - → 直接解决障碍，干就完了。

#### ⑥ 制定具体、可落地的目标

把大类变成**可衡量**的目标：

```
我想要【这里填入你的大类名称】达到【可衡量 / 数字 / 状态】的程度。
```

#### ⑦ 拨出具体时间

不给目标安排时间，它永远只会停留在纸上。

例子：
- 每天早上 8:00–8:30 做这件事
- 每天刷完牙后，做 10 分钟
- 每周六下午 3:00–4:00 固定执行

## 两种模板（直接套用）

到这一步，只有两种结果：

**要么真心想做，要么果断划掉。**

我给你写好了模板，直接填空就行。

### 模板 A：我真的想要，继续做

```
过去我以为想要【这里填入你的大类名称】。

逻辑理由：……

情绪理由：……

我确定：我是真的想要。

障碍：……

分析：…… 在我的掌控中。

具体目标：我想要【这里填入你的大类名称】达到【状态 / 数字】。

固定时间：【具体时间段】

现在：

我想要【这里填入你的大类名称】，因为【情绪理由】。

我的具体目标是【可衡量目标】。

我会在【固定时间】认真执行。
```

### 模板 B：我其实不想要，果断划掉

```
过去我以为想要【这里填入你的大类名称】。

逻辑理由：……

情绪理由：……

我意识到：我根本不是真的想要。

现在：

我已经把【大类】从我的时间表和待办清单中划掉。

我以前只是因为【理性理由】才以为自己想做，其实内心并不渴望。

从今以后，我不再为它内耗，我要心无负担地，只做真正想做的事。
```

## 举两个我自己的例子

### 我真的想要，继续做

过去我以为想要【学习与人良好沟通】

- 逻辑理由：因为与人沟通良好可以让我交到新的朋友
- 逻辑理由：因为与人沟通良好可以让我和客户沟通更通畅
- 情绪理由：因为不会与人沟通给我带来了不好的体验，我再也不想有那样的体验了

我的真心：我确实希望能够跟人好好沟通。

障碍：
- 没有花时间观察学习擅长沟通的人是如何沟通的
- 没有花时间听课和看书
- 学了一些技巧，但从没应用过

分析障碍
- 在我的掌控中
- 在我的掌控中
- 在我的掌控中

具体目标：我想要【与人沟通】达到【心里不慌，顺利地开始和结束对话】的程度。

具体时间：每天早上 8 点到 8 点半看《人性的弱点》这本书，同时做笔记，并且在工作中可以应用的时候应用学到的知识。

现在：我想要【学习与人良好沟通】，因为【不会与人沟通给我带来了不好的体验，我再也不想有那样的体验了】，我的具体目标是【与人沟通达到心里不慌，顺利地开始和结束对话的程度】。我准备【每天早上 8 点到 8 点半】看《人性的弱点》这本书，同时做笔记，并且在工作中可以应用的时候应用学到的知识。

### 我其实不想要，果断划掉

过去我以为想要【学骑马】

- 逻辑理由：骑马是一种技能，技多不压身
- 逻辑理由：如果哪天车不能用了，可以骑马
- 逻辑理由：新疆马多，学起来很方便
- 情绪理由：看着很酷

我的真心：我觉得能骑马肯定很酷，但我根本用不着。只是因为新疆马多，所以我觉得学也可以。其实我根本就没多想学。我要把【学骑马】从我的时间表和待办清单上划掉。

现在：我意识到我根本不想要 *【学骑马】*，我已经把它从我的时间表和待办清单上划掉了。我以前因为 *【学起来方便】* 这样的理由，才以为自己想要做这件事的，其实我根本不想做。以后我再也不想这件事了，我要心无负担地做我真正想做的事。

## 最后我想说

我强烈建议你把这份分析完整保存下来。

人很容易忘记初心，做着做着累了、烦了，就容易放弃。

每当你动力不足时，回头看看自己亲手写的分析，会立刻重新找到方向。
