关于我

我是向西，目前在一家外企任职，有10年工作经验。

好奇心旺盛，喜欢探索世界。曾在7座城市旅居，平时喜欢读书、玩桌游、打羽毛球。

习惯定期复盘个人经历，致力于在反思中不断提升自己。以此为契机，每周六会在本站更新一篇日常思考，解决一个问题。

我的独立项目

🧪 Smile Lab

项目链接：Smile Lab

同样是以游戏的形式训练正向偏向。这个项目和Smile Finder的区别是，这个项目用的是真人图片，所以训练效果指数增强。

我在这个项目上付出了很多心血，也学到了很多。此项目正在火热众筹中。

😊 Smile Finder

项目链接：Smile Finder

为了度过我那疼痛的20多岁，我一直在练习正向偏向训练（练习从一堆表情中找出笑脸）。2026年，我创建了自己的正向偏向训练网站，希望能以游戏的形式，让训练正向偏向这件事变得更简单。

这个网站就是Smile Finder。

如果你觉得自己也需要思想更加正向，请随时光顾，手机电脑随时随地来一局！

📊 SQL Wizland

项目链接：SQL Wizland

在日常工作中，如果遇到某一句 SQL 语法忘掉了，搜索一下就可解决。但这样的结果是，面试前，总要复习复习再复习。

为了让自己复习的时候，能够在场景里复习，在实际业务中复习，我创建了帮助自己复习 SQL 语法的网站：SQL Wizland。如果你也有同样的困扰，欢迎访问。里面包含2道苹果面试题哦。

我的生活复盘

网购好东西

我实在讨厌网购刷来刷去浪费时间，所以从2022年开始记录我网购过的所有商品。好的记录在案以后直接买，持续踩坑的店直接拉黑。

现在我的购物变得极度高效、省心。所以把我的购物推荐总结在本站，想要省时、省事、不踩坑？直接买！（全部都是本人买过的商品，请放心购买）。

旅居

身体和灵魂总要有一个在路上？成年人当然两个都要！

借着在家办公的机会，从2022年起，我在7座从未去过的城市旅居，得以脱离游客视角细细领会各地生活。我相信：冒险可能会受伤，但单调是会死人的。

如果你也想要旅居，可以参考我的旅居系列文章，希望能给你带来灵感。

读书

想要提升自己，看书是无法避免的。我把读书看做和伟人对话的机会。从2011年本科毕业后，每年都读很多书。涉猎的话题也极为广泛。

如果想要看看我从这些对话中学到了什么，请随时查看我的阅读感悟。

热爱敦煌

为了摆脱在佛教方面知识=0的无知状态，2021年起我踏上了研究佛教的不归路。我把自己一路走来用过的资源总结到了这篇零基础佛教入门指南里。

如果你也想要从看不懂壁画到能够一眼辨别年代，欢迎加入研究队伍。
