Chrome网页怎么长截图？2个超简单方法：原生指令vs国内可用插件（2026实测有效）

By 向西
TOOLS RESOURCES CHROME EXTENSION TECH WORKPLACE

文章目录

你遇到过这种需求吗？要你把一整个网页完整截下来。

你是怎么做的？

移动鼠标、使用截图软件，一页一页地截下来，然后再用 Word 把所有图片拼起来？

今天分享两个解放你双手的工具，让你2秒搞定完整网页内容的截图。

没错，普通截图确实只能截屏幕看得见的部分，一长就废。

但是，今天分享的两个我最常用、最简单的Chrome全页截图方法能帮你解脱：

  • 一个只需下载插件即可，超级方便（国内也可以用）；
  • 另一个连插件都不需要，只要浏览器就行！（公司电脑不让装插件？放心用）

Chrome 浏览器 Photo by AS Photography

使用 Chrome 扩展（最简单、最推荐）

  • 插件名：Go Full Page
  • ID：fdpohaocaechififmbbbbbknoalclacl
  • Chrome web store link：https://chromewebstore.google.com/detail/gofullpage-full-page-scre/fdpohaocaechififmbbbbbknoalclacl

这个插件不止允许一键截取整页，而且既可以下载成图片也可以下载成 PDF。

使用有多简单呢，点一下插件图标，当前页面就会自动开始滚动，然后你就可以下载到本地了。

超级无脑，小白一秒上手。

因为国内访问不了 Chrome web store 想放弃？大可不必，有解决方案，请查看这篇《国内谷歌 Chrome 浏览器扩展程序下载安装指南》，里面有详细的下载和安装教程，让你以后再也不必妥协放弃。

公司电脑不能装插件？用原生方法！

但如果你们公司的电脑禁止安装扩展怎么办呢？

没关系。Chrome 自带隐藏全页截图功能，白嫖就行。

操作步骤

  1. 打开要截图的页面
  2. 打开开发者工具
    • Mac：Command + Option + I
    • Windows：Fn + F12
  3. 打开命令菜单
    • Mac：Command + Shift + P
    • Windows：Control + Shift + P
  4. 输入： full
  5. 会出现唯一一个选项： Capture full size screenshot
  6. 按回车
  7. 自动下载完整长截图

搞定！

不用装任何东西，公司电脑也能放心用。

如果你也经常需要长截图， 这两个方法一定要收藏， 真的能省超多时间！
